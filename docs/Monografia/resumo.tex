Neste projeto, é desenvolvido um robô-guia para ambientes internos chamado SAURON. O sistema é composto por dois módulos, localização e navegação. Localização significa estimar a postura do robô em relação a uma representação global do ambiente em que se encontra. A especificação de uma abordagem de localização baseada em dados sensoriais possui uma estimativa inicial da postura do robô e usa os dados coletados pelos sensores, juntamente com os dados que podem ser obtidos de uma mapa do ambiente, para produzir uma estimativa precisa da postura. Uma dificuldade é que os dados sensoriais são corrompidos por erros de medidas, decorrentes, principalmente, de ruído. O uso de dados originados em múltiplos sensores fornece informação redundante e complementar, que pode ser utilizada para gerar uma estimativa combinada, aumentando a confiança na postura calculada. Este trabalho versa sobre o desenvolvimento de um robô móvel de baixo custo, equipado com odômetro, um semi-anel contendo oito sonares e uma câmera de vídeo convencional, capaz de guiar pessoas a regiões específicas de um ambiente, de acordo com o destino que lhe for indicado. As estimativas de postura são calculadas a partir de um Filtro de Kalman Estendido (EKF). O SAURON é capaz de navegar intermapas, de modo que possa percorrer mais de um andar de um edifício. A busca de rota intermapa é realizada através de uma busca em profundidade, ao passo o A* é utilizado para rotas intramapa. Os resultados obtidos foram satisfatórios. O robô foi capaz de atingir seus destinos com erros abaixo de 50 cm, valor aceitável para os propósitos do projeto.
