Um componente essencial no sistema de um robô móvel é a habilidade de se localizar com acurácia, o que envolve estimar sua postura em relação a uma representação global do ambiente em que se encontra. A especificação de uma abordagem de localização baseada em dados sensoriais possui uma estimativa inicial da postura do robôs usa os dados coletados pelos sensores, juntamente com os dados que podem ser obtidos de uma mapa do ambiente, para produzir uma estimativa precisa da postura. Uma dificuldade é que os dados sensoriais são corrompidos por erros de medidas, decorrentes, principalmente, de ruído. O uso de dados originados em múltiplos sensores fornece informação redundante e complementar, que pode ser utilizada para gerar uma estimativa combinada, aumentando a confiança na postura calculada. Este trabalho versa sobre o desenvolvimento de um robô móvel de baixo custo, equipado com odômetro, um semi-anel contendo oito sonares e uma câmera de vídeo convencional, capaz de guiar pessoas a regiões específicas de um ambiente, de acordo com o destino que lhe for indicado. As estimativas de postura são calculadas a partir de um Filtro de Kalman Estendido (EKF).
